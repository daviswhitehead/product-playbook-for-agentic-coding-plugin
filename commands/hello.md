# Hello from Product Playbook

You have successfully installed the **Product Playbook for Agentic Coding** plugin!

## Available Commands

This plugin provides a structured 4-phase workflow for agentic software development:

| Command | Phase | Description |
|---------|-------|-------------|
| `/playbook:product-requirements` | Product Discovery | Define WHAT to build and WHY |
| `/playbook:tech-plan` | Solution Planning | Design HOW to build it |
| `/playbook:tasks` | Delivery | Break work into actionable tasks |
| `/playbook:work` | Delivery | Execute the next task |
| `/playbook:debug` | Utility | Systematic debugging workflow |
| `/playbook:learnings` | Retrospective | Capture and document learnings |

## Quick Start

1. Start a new project with `/playbook:product-requirements`
2. Plan the solution with `/playbook:tech-plan`
3. Break down tasks with `/playbook:tasks`
4. Execute work with `/playbook:work`
5. Capture learnings with `/playbook:learnings`
